# 70 MCQs on OpenAI Agents SDK: Runner Methods and Streaming
1. **What is the primary purpose of the `runner.run()` method in the OpenAI Agents SDK?**  
   a) To stream responses in real-time  
   b) To send a query and receive a complete response at once  
   c) To configure the agent’s model

2. **What is the nature of the `runner.run()` method?**  
   a) Synchronous  
   b) Asynchronous  
   c) Both synchronous and asynchronous  
   d) Neither synchronous nor asynchronous

## Questions on Error Handling

51. **What error occurs if you call `runner.run()` without awaiting it?**  
    a) `RuntimeError: no running event loop`  
    b) `AttributeError: 'coroutine' object has no attribute 'final_output'`  
    c) `GuardrailTripwireTriggered`  
    d) `MaxTurnsExceeded`  
    **Answer**: b) `AttributeError: 'coroutine' object has no attribute 'final_output'`  

52. **Why does the error `RuntimeError: This event loop is already running` occur?**  
    a) When `runner.run_sync()` is called inside an async function  
    b) When `runner.run()` is called without `await`  
    c) When the API key is invalid  
    d) When the context is missing  
    **Answer**: a) When `runner.run_sync()` is called inside an async function  

53. **What error occurs when calling `runner.run_streamed()` without an event loop?**

    b) `RuntimeError: no running event loop`  
    c) `GuardrailTripwireTriggered`  
    d) `MaxTurnsExceeded`  
    **Answer**: b) `RuntimeError: no running event loop`  

54. **What does the `MaxTurnsExceeded` exception indicate?**  
    a) The agent’s context is invalid  
    b) The maximum number of agent turns has been exceeded  
    c) The API key is missing

55. **What does the `GuardrailTripwireTriggered` exception indicate?**

    b) A guardrail condition has been violated  
    c) The API key is invalid
